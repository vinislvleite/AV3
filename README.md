# AV3 - Sistema Aerocode

Terceira Avaliação Individual - Engenharia de Software

## Descrição
Sistema web desenvolvido para o gerenciamento de produção de aeronaves, abrangendo cadastro de peças, etapas de produção, testes de qualidade e geração de relatórios.

## Tecnologias Utilizadas
- Frontend: React e Vite
- Backend: Node.js e Express
- Banco de Dados: MySQL e Prisma ORM
- Linguagem: TypeScript

## Como rodar o projeto

Siga os passos abaixo para executar a aplicação localmente.

### 1. Clonar o repositório

### 2. Configurar o Backend
Abra um terminal, acesse a pasta do servidor e instale as dependências:

cd backend<br>
npm install<br>
npm run dev<br>

(O servidor rodará na porta 3333 e fará a configuração inicial do banco de dados automaticamente).

### 3. Configurar o Frontend
Abra um segundo terminal, acesse a pasta da interface e instale as dependências:

cd frontend<br>
npm install<br>
npm run dev<br>

(A interface rodará na porta 5173).
